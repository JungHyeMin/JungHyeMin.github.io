Pycaret 설치

  1. 명령프롬프트를 관리자로 실행

  2. 바탕화면에 pycaret 폴더 생성

  3. 명령프롬프트에서 cd로 해당 폴더로 이동

-> cd C:\Users\1\Desktop\pycaret

  1. 가상 환경 만들기
    $conda create –name pycaret(이름설정) python=3.8

  2. 가상환경 접속
    $conda activate pycaret

  3. pycaret 설치
    $ pip install pycaret

  4. 설치 오류가 발생

아래 링크로 접속

“https://visualstudio.microsoft.com/visual-cpp-build-tools/
